INTRODUCTION
------------

This module doesn't do anything on its own, just provides functions for the
modules cfde and cfdr

INSTALL
-------

The module needs 2 files in directory 'resources/sat/':

cfdv2.xsd .................. [1]
cadenaoriginal_2_0.xslt  ... [2]

End result should be for example: 'resources/sat/cfdv2.xsd'

Available in:
[1] http://www.sat.gob.mx/sitio_internet/cfd/2/cfdv2.xsd
[2] http://www.sat.gob.mx/sitio_internet/cfd/2/cadenaoriginal_2_0/cadenaoriginal_2_0.xslt

MORE INFO in spanish in README.ES.txt
