This project aims to provide a node content type for CFD (Digital Fiscal Proof - in Spanish) which is basically an electronic bill (a CFD is in fact a XML file).

Drupalers in Mexico may find this project useful.

The project containts 3 modules: cfd, cfde, and cfdr:
cfd - this module only provides common functions between cfde and cfdr
cfde - intented for distribution (emision) of CFDs
cfdr -intended for access to received CFDs
